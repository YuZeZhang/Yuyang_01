需要安装的插件列表：

1 element ui

```
npm i element-ui -S
```

2 axios

```
npm i axios -S
```

3 vue-table-with-tree-grid(树表)

```
npm i vue-table-with-tree-grid -S
具体文档可参考：
https://github.com/MisterTaki/vue-table-with-tree-grid
```

4 js-cookie

```
npm i js-cookie -S
```

5 gojs

```
npm i gojs -S
```
